# 📝 To-Do List Web App | CodeAlpha Internship Task

A responsive and interactive To-Do List web application built using HTML, CSS, and JavaScript to help users manage daily tasks efficiently.

---

## 🚀 Live Features

- ✅ Add new tasks instantly  
- ✏️ Edit existing tasks  
- 🗑 Delete tasks  
- ✔ Mark tasks as completed  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Layout  
- **JavaScript (ES6)** – Functionality & Logic
